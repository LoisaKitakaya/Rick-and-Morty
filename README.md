# rick-and-morty

## Project setup

To run this project locally:

```
git clone https://github.com/LoisaKitakaya/Rick-and-Morty.git

cd Rick-and-Morty

npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run deploy
```
